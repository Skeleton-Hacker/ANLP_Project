# ANLP_Project

## Installing the enviornment
Use micromamba to create and activate the enviornment the file can be found in environment.yml. 
micromamba create -f environment.yml --channel-priority flexible
Do install bert-score using pip
## Running Code
The folder pipeline contains the code to run the mini lm v6 while pipeline_2 contains the code for the BGE embeddings. Refer to `run.md`
